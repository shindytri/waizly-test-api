This API contains the endpoint for user management. This API can handle:
1. User registration
2. Get all user data
3. Get user data based on the user id
4. Update user data based on the user id
5. Delete user data based on the user id
Authorization used in this API are 'Basic Auth', where the default credentials are available in 'component/glocalVariables.js'

Tech Stack used for this API are:
1. MongoDB (Currently the API are connected to local database)
2. NodeJS
Optional:
Postman to test the API

How to run the API:
1. Clone the API repository
2. run 'npm install'
3. run 'node index.js'
